---
description: Initialize Yoloop in the current project
argument-hint: "GOAL"
allowed-tools: Bash(yoloop:*)
---

Run the Yoloop initializer in the current project.

Use Bash to run:

```sh
yoloop init --goal "$ARGUMENTS"
```

After initialization, tell the user to edit `GOAL.md`, `PLAN.md`, and `TASKS.json`, then run `yoloop doctor`.
